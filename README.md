## 🚀 Getting Started (Local Development)

This portfolio is built using **Astro**. To run it locally:

### Prerequisites

Make sure you have Node.js (v18.x or higher) and one of the package managers installed (`npm` or `pnpm`).

### 1. Clone the repository

```bash
git clone https://github.com/Nico4652/Portafolio.git
cd Portafolio
```

### 2. Install dependencies

```bash
# If using npm:
npm install

# If using pnpm:
pnpm install
```

### 3. Run the development server

```bash
# If using npm:
npm run dev

# If using pnpm:
pnpm run dev
```

Open `http://localhost:4321` in your browser to view the site.

### 4. Build for production

```bash
# If using npm:
npm run build

# If using pnpm:
pnpm run build
```

The output will be generated inside the `/dist` directory.
